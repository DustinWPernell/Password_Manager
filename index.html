Password Manager

Maintained by Dustin Pernell

=================================================================================================


The Application

Password Manager iPhone/Android app stores passwords and security questions for easy and protected access. This project was started Dustin Pernell as a final project for an Android development class. Dustin then developed the iPhone version later as another project for school.

=================================================================================================


Features
Password database
- Stores passwords in an encrypted local database.

Security question database
- Stores security questions in an encrypted local database.

Biometric authentication
- Uses built in biometric authentication to access encrypted data.

Usage Instructions

Android
Step 1: The first thing you will see when launching the application is the

iPhone
Step 1:
Launching the App 		The first thing you will see when launching the application is the main screen.
You will need to select login.

The next screen will cause the application to crash when entering a password.
It is recommended to bypass this screen by modifying the Main.storyboard to start at the navigation controller labeled Title.
Step 2:
Add an Account 		Next you will add an account using the + button in the top right of the Account List view.

The sample image shows an account label as "Google".
Step 3:
Fill out Account 		You will need to fill out the account information.

You may add security questions by selecting "View security Questions".
Step 4:
View an Account 		When selecting an account from the Account List you will see the details of that account.
You can edit the account by clicking "Edit".

Selecting "View Security Questions"
 This allows you to view the security questions associated with that account.
Selecting "View Account History"
 This will show you the changes that have been made to that account.

FAQs

Does this application use encryption to save the passwords?
- The Android application does use encryption.
- The iPhone does not as it is not finished and that was the last feature designated to be implemented.

Will the applications be put onto either app store for android or iPhone/
- As of July 2020, there are no plans to publish the applications the either app store.
